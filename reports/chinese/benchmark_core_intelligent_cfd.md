## 不能回避的三个盲区

1. **“神经网络”不是智能 CFD 同义词。** 计算力学学报的 79 条中包含结构、岩土等方向；若不做领域上下文二次筛选，结果数量越大，假阳性越多。
2. **“每日新增”不能用期刊发表日期判断。** 月刊/半月刊会集中上线；系统应以 `first_seen`（首次发现）作为日报新增口径，保留 `published` 作为来源发表日期。
3. **自动化检索不等于可回溯全库。** 知网单次结果页默认只展示 20 条，且会验证码/登录态失效。历史全量回溯必须支持分页或知网导出 RIS/EndNote 后的半自动导入；否则不能宣称“全库覆盖”。

## 推荐的分层查询

对每本核心期刊分别运行下列主题式，而不是只搜“神经网络”。每组再与期刊名做“文献来源（精确）”组合：

1. `(物理信息神经网络 + 神经算子 + 深度学习 + 机器学习) * (流场 + 流动 + 湍流 + 气动 + 空气动力学)`
2. `(代理模型 + 降阶模型 + 自动编码器 + 多保真 + 迁移学习) * (CFD + 流场 + 翼型 + 绕流)`
3. `(强化学习 + 数据驱动 + 图神经网络 + 自监督) * (流动控制 + 流场重构 + 可压缩流 + 多相流)`

用这三组的并集做候选池，再由项目的严格智能 CFD 规则去重、排序和生成日报。
